# ChatGPT Clone V2

This is version 2 of the ChatGPT clone ([supakit_chatgpt](https://github.com/programmingenjoyer219/supakit_chatgpt)), built using :

- [SvelteKit (Svelte 5)](https://svelte.dev/)
- [Convex](https://www.convex.dev/)
- [Auth.js](https://authjs.dev/)
- [Gemini API](https://ai.google.dev/)
- [TailwindCSS](https://tailwindcss.com/)
- [TypeScript](https://www.typescriptlang.org/)
- [Svelte-French-Toast](https://svelte-french-toast.com/)
- [Remix Icons](https://remixicon.com/)
